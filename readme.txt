Proposez à vos visiteurs des séances pédagogiques prêtes à intégrer dans monCahierJournal.com.



== Description ==



Ce plugin vous permet d’ajouter un rapidement un cours prêt à intégrer dans [monCahierJournal.com](http://monCahierJournal.com).



Il affichera à votre visiteur une icône [monCahierJournal](http://monCahierJournal.com) qui enverra le cours tel que vous 
l’aurez préparé dans son Cahier-Journal.



== Installation ==



1. Envoyez ‘moncahierjournal.php’ dans le répertoire ‘/wp-content/plugins/moncahierjournal’

1. Activez le plugin via le menu ‘Extensions’ de WordPress

1. Ecrivez le code ‘[monCahierJournal nom="Titre du cours" sequence="Titre de la séquence" objectif="&lt;p&gt;Contenu pédagogique&lt;/p&gt;"]’ dans vos Articles ou dans vos Pages.



== Configuration ==



Les différentes options du shortcode sont les suivantes :



= nom =


Le titre de la séance : texte brut. Pas de mise en page possible.



= sequence =


Le titre de la séquence : texte brut. Pas de mise en page possible.

Attention : pas d’accent à "sequence". Le texte, lui, peut contenir des accents.



= objectif =


Le contenu pédagogique de la séance. Texte brut ou formaté en html. 


Note : Les balises autorisées sont les suivantes : p,u,b,i,strong,em,br,ul,ol,li,a (les seuls attributs possibles sont href="http://exemple.com" et target="_blank").
